# Pirate Ship Inventory Management System

This project is a JavaFX-based inventory management system designed to help manage stock on a pirate ship. 
All crewmates can login, add stock items, and view current inventory. 
Certain actions, like viewing stock changes, are restricted to specific roles such as the Quartermaster.
